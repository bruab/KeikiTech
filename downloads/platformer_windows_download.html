KeikiTech | Running Platformer on Windows

Students from the June Game Development Workshop can now download and play their own modified version of the game.

Downloads:

  • Python 3.3 for Windows
  • Platformer code

Instructions:

  • Download both of the files above.
  • Unzip them. You can put them anywhere.
  • To play Platformer, drag the Platformer-master/main.py file and drop it on the Python33/python.exe file.
  • Contact us if you have any problems. Have fun!
